# AI Vendor Sourcing Agent

A copy of the end-to-end procurement demo from Vercel, rebuilt as a runnable vinext app.

## Included flows

- Procurement intro page
- Mock persona login
- Purchase manager dashboard
- Sample vendor and quotation data
- Quotation comparison and approval interaction
- Templates, notifications, demo console, production, and QC routes

## Run locally

```bash
npm install
npm run dev
npm run build
npm run lint
```

The project uses browser-local demo state and does not require a database or external credentials.
